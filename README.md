# CS449 Peg Solitaire Project

This repository contains my Peg Solitaire project for COMP-SCI 449.

## Sprint 0
Tool setup, GitHub repository creation, and basic GUI and unit test practice.

## Sprint 2
Implementation of the core Peg Solitaire components.

### Files
- board.py – Handles the board structure and peg positions
- game_logic.py – Implements the rules and move validation
- gui.py – Tkinter graphical interface for the game
- main.py – Runs the application
- test_game.py – Unit tests for the game logic

## How to Run the Program
1. Install Python.
2. Open the project in Visual Studio.
3. Run `main.py`.

## Features
- Graphical Peg Solitaire board
- Move validation
- Unit testing for game logic
